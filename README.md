# AV.SC.U4AIE23151

Campus notification platform with priority inbox.

## Structure

logging_middleware/     - Reusable logging middleware 
notification_system_design.md  - Stages 1-6 design
notification_app_be/    - Backend 
notification_app_fe/    - Frontend 

### Backend
cd notification_app_be
npm install
npm run dev
Runs on http://localhost:4000

### Frontend
cd notification_app_fe
npm install
npm run dev
Runs on http://localhost:3000
