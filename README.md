# AV.SC.U4AIE23151

Campus notification platform with priority inbox.

## Structure

logging_middleware/     - Reusable logging middleware <br>
notification_system_design.md  - Stages 1-6 design <br>
notification_app_be/    - Backend <br>
notification_app_fe/    - Frontend <br>

### Backend
cd notification_app_be <br>
npm install <br>
npm run dev <br>
Runs on http://localhost:4000 <br>

### Frontend
cd notification_app_fe <br>
npm install <br>
npm run dev <br>
Runs on http://localhost:3000 <br>
